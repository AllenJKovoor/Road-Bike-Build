# 🚴 AI-Assisted Road Bike Build
Documenting the research, design decisions, and build process for a custom road bike using Chinese components, leveraging AI-assisted prompting and structured problem solving.

## About

Personal project by Allen — Computer Science graduate exploring the 
intersection of structured AI-assisted research and real-world engineering 
problem solving.

## Overview
This repository documents the end-to-end process of researching, designing, 
and building a custom road bike using Chinese-manufactured components. 

Rather than following a traditional build guide, this project uses AI as a 
structured research and decision-making tool — treating each major build 
decision as a problem to be broken down, researched, and solved systematically.

---

## Why This Project

Road bikes from major Western brands carry significant brand markup. Chinese 
manufacturers produce components to the same specifications at a fraction of 
the cost — but navigating the market requires careful research into quality, 
compatibility, and value. This project documents that research process in full.

---

## How AI Is Used in This Project

AI is used as a research and reasoning assistant throughout this build. 
Rather than using it to generate generic answers, each prompt is structured 
around a specific decision or problem — such as frame geometry selection, 
component compatibility, or groupset comparisons.

The goal is to use AI the way an engineer would use any tool — deliberately, 
critically, and with clear intent. All key prompts and AI-assisted decisions 
are documented in the `/prompts` folder.
